# AlrightOS-Bot
AlrightOS's offical discord bot.

## How to install it?

```
//clone repo
$ git clone https://github.com/batuhantrkgl/AlrightOS-bot

//cd repo
$ cd AlrightOS-bot

// install packages
$ npm install

//start the bot
$ node .
```

## Configration Files
`` config.json & deploy-commands.js ``
